# Media Library
## C\# Practice Project
A C\# Console App Media Library with classes that stores Audio Books, Albums, Books, eBooks, and Movies
### Audio Books
- Title
- Author
- Publisher
- Publish Date
- Length
- Format
- Genre
- Performed By
### Albums
- Title
- Artist
- Release Date
- Label
- Format
- Genre
### Books
- Title
- Author
- Publisher
- Publish Date
- Page Count
- Format
- Genre
### eBooks
- Title
- Author
- Publisher
- Publish Date
- Page Count
- Format
- Genre
### Movies
- Title
- Director
- Release Date
- Actors
- Producers
- Editor
- Studio
- Genre
- Length
